# Agent Entry

## Purpose

这是一个已接入 Project Ledger 的既有项目。Project Ledger 只用于帮助后续用户和 agent 快速理解项目边界、当前状态、运行线索和最近上下文。

## Read First

- `project_ledger/03_PROJECT_CARD.md`
- `project_ledger/04_STATUS.md`
- 最近的 `project_ledger/runs/*.md`

## Read If Needed

- 如需了解接入来源，再读 `project_ledger/02_ADOPTION_NOTE.md`。

## Working Rules

- 不要擅自扩大范围。
- 不要把待确认内容当成事实。
- 不要每轮全量读取或重写 `project_ledger/`。
- 普通任务只写短 run note。
- 状态变化才更新 `04_STATUS.md`。
- 重大取舍才更新 `06_DECISION_LOG.md`。
